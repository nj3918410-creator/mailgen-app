# Mailgen Android APK - GitHub Actions Build

## ✅ এই ZIP দিয়ে FREE APK বানানোর সহজ ধাপ:

---

## ধাপ ১ — GitHub Account খুলুন
- https://github.com এ যান
- Sign Up করুন (বিনামূল্যে)

## ধাপ ২ — নতুন Repository তৈরি করুন
1. GitHub-এ লগিন করুন
2. উপরে ডানে "+" বোতামে ক্লিক করুন
3. "New repository" সিলেক্ট করুন
4. Repository name: `mailgen-app`
5. "Public" সিলেক্ট করুন
6. "Create repository" ক্লিক করুন

## ধাপ ৩ — ফাইলগুলো Upload করুন
1. Repository-তে "uploading an existing file" ক্লিক করুন
2. এই ZIP এর ভেতরের **সব ফাইল ও ফোল্ডার** drag করে upload করুন
3. "Commit changes" ক্লিক করুন

## ধাপ ৪ — GitHub Actions চালু করুন
1. Repository-তে **"Actions"** ট্যাবে যান
2. "Build APK" workflow দেখবেন
3. **"Run workflow"** বোতামে ক্লিক করুন
4. আবার "Run workflow" ক্লিক করুন

## ধাপ ৫ — APK Download করুন
1. ২-৫ মিনিট অপেক্ষা করুন
2. Build শেষ হলে সবুজ ✓ দেখাবে
3. Build-এ ক্লিক করুন
4. নিচে **"Artifacts"** সেকশনে **"Mailgen-APK"** দেখবেন
5. ক্লিক করুন — APK download হবে!

---

## ⚠️ সমস্যা হলে:
- Actions ট্যাব দেখা না গেলে: Settings > Actions > Allow all actions
- Build fail হলে: Actions log দেখুন, সাধারণত Gradle version issue

## 📱 APK Install করুন:
1. Android ফোনে APK ফাইল পাঠান
2. Settings > Security > Unknown Sources চালু করুন  
3. APK ফাইলে ক্লিক করে Install করুন
